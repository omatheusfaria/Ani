# Regras de Negócio

Este documento descreve restrições e comportamentos que orientam a lógica principal de **Ani: A Jornada da Virtude**.

| Código | Nome | Descrição |
| --- | --- | --- |
| RN001 | Registro de memória única | Cada documento encontrado é registrado apenas uma vez no diário. Revisitas podem desbloquear variantes. |
| RN002 | Ordem variável de memórias | As memórias podem aparecer em ordem variável entre sessões, reforçando a singularidade de cada jornada. |
| RN003 | Efeito imediato de escolha | Toda escolha moral deve gerar efeito visual e sonoro imediato, além de persistir no estado do mundo. |
| RN004 | Critério de desbloqueio | Um novo cômodo só pode ser desbloqueado após o cumprimento dos critérios narrativos definidos para aquele conjunto de memórias. |
| RN005 | Ansiedade e consequência | Falhar no mini-jogo de respiração aumenta distorções e pode bloquear opções sociais em determinados momentos. |
| RN006 | Conflito de escolhas | Quando houver revisita de uma mesma memória com nova decisão, o sistema deve considerar apenas a escolha válida mais recente. |
| RN007 | Dados de avaliação | Logs de escolhas e capturas podem ser exportados localmente para análise acadêmica ou testes internos. |
| RN008 | Conteúdo sensível | O jogo deve oferecer opção de pular cenas sensíveis sem quebrar a continuidade da narrativa. |
