# Requisitos Funcionais

Este documento reúne as principais funcionalidades esperadas para **Ani: A Jornada da Virtude**.

| Código | Título | Categoria | Prioridade | Descrição |
| --- | --- | --- | --- | --- |
| RF001 | Movimentar personagem | Evidente | Altíssima | O sistema deve permitir que o jogador mova Ani lateralmente usando teclado. |
| RF002 | Pular | Evidente | Altíssima | O sistema deve permitir que Ani realize pulos básicos para superar obstáculos. |
| RF003 | Colisão com cenário e objetos | Evidente | Altíssima | O sistema deve impedir que Ani ultrapasse paredes, pisos e objetos sólidos. |
| RF004 | Interagir com objetos | Evidente | Altíssima | O sistema deve permitir interação com objetos significativos para abrir memórias. |
| RF005 | Sistema de memórias | Evidente | Altíssima | O sistema deve apresentar memórias como cenas ou documentos que podem ser lidos e registrados no diário. |
| RF006 | Sistema de escolhas morais | Evidente | Altíssima | Ao enfrentar dilemas, o jogador deve escolher entre virtude, vício menor ou vício maior. |
| RF007 | Atualização visual do ambiente | Evidente | Altíssima | O sistema deve atualizar cores e sons do ambiente imediatamente após uma escolha moral. |
| RF008 | Inventário de documentos | Evidente | Altíssima | O sistema deve registrar todos os documentos em um diário acessível ao jogador. |
| RF009 | Desbloqueio progressivo do cenário | Evidente | Altíssima | Novos cômodos e portas devem surgir conforme memórias são exploradas. |
| RF010 | Sistema de ansiedade | Evidente | Alta | O sistema deve oferecer um mini-jogo de respiração em crises de ansiedade. |
| RF011 | Sistema de puzzles | Evidente | Alta | O sistema deve incluir puzzles contextuais, como reconstruir fotos e decifrar cartas. |
| RF012 | Diálogos com NPCs | Evidente | Alta | O sistema deve apresentar diálogos textuais fragmentados com NPCs que representam vozes do passado. |
| RF013 | Sistema de save/load | Evidente | Altíssima | O sistema deve salvar e carregar o progresso do jogador, incluindo memórias, escolhas e estado da casa. |
| RF014 | Feedback sonoro e musical | Evidente | Alta | O sistema deve adaptar trilha sonora e efeitos conforme o estado emocional e visual da experiência. |
| RF015 | Suporte a gamepad | Evidente | Média | O sistema deve permitir que o jogo seja controlado também por gamepad. |
